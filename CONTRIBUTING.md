# Contributing

## Bug reports

Open a GitHub Issue. Include fossic version, OS, Rust version, and a minimal reproducer.

## Pull requests

- Describe what the PR changes and why. Reference an issue if applicable.
- Run `cargo fmt` and `cargo clippy --all-features -D warnings` before submitting.
- Run `just test` — all tests must pass.
- For API changes, update `docs/architecture.md` and `docs/operating.md` to match.

No CLA required. No formal code of conduct yet.
